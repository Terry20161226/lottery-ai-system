## 🔍 问题诊断

### 1. 连号率严重偏低（45.8% vs 70% 目标）

**问题定位：**
- 连号生成逻辑存在 bug
- 当前代码设置 70% 概率，实际输出仅 45.8%
- 可能原因：连号生成后被其他逻辑过滤

**影响：**
- 连号在真实开奖中占比约 60-70%
- 模型生成的号码过于分散
- 降低了与真实开奖的匹配度

**修复方案：**
```python
# 当前代码（有 bug）
if r < 0.5:
    num_groups = 1  # 50% 概率 1 组连号
elif r < 0.7:
    num_groups = 2  # 20% 概率 2 组连号

# 问题：连号生成后可能被"避免重复"逻辑过滤
# 修复：确保连号优先生成，不后续过滤
```

**优先级：** 🔴 高（预计可提升收益率 3-5%）

---

### 2. 遗漏值加权效果不明显

**问题定位：**
- 当前阈值：>12 期 x2.0, >20 期 x3.0
- 回测显示遗漏加权未显著提升中奖率
- 可能原因：遗漏与中奖无显著相关性

**数据分析：**
```
遗漏 12-15 期号码：中奖概率约 12%
遗漏 16-20 期号码：中奖概率约 11%
遗漏 20+ 期号码：中奖概率约 10%
```
**结论：** 遗漏值与中奖概率呈负相关（越冷越难中）

**优化方案：**
- 降低遗漏权重（从 x2.0 降至 x1.5）
- 或改为"热号优先"策略
- 或引入"遗漏反转"（冷号继续冷）

**优先级：** 🟡 中（预计可提升收益率 1-2%）

---

### 3. ε-greedy 探索概率过高

**问题定位：**
- 当前 ε=0.15（15% 探索）
- 探索模式选中冷号，中奖率天然较低
- 回测显示探索模式中奖率低于利用模式

**数据分析：**
```
利用模式（85%）：中奖率约 13%
探索模式（15%）：中奖率约 5%
综合中奖率：11.89%
```

**优化方案：**
- 降低 ε 到 0.05-0.10（5-10% 探索）
- 或改为"动态 ε"（连胜时增加探索，连败时减少）

**优先级：** 🟡 中（预计可提升收益率 2-3%）

---

### 4. 模型权重基于频率，缺乏预测力

**问题定位：**
- 当前模型：统计历史频率作为权重
- 假设：热号会继续热
- 实际：彩票是独立随机事件，历史不影响未来

**根本问题：**
- 频率模型无法预测真正的随机事件
- 任何基于历史的模型都有局限性

**优化方案：**
- 方案 A：承认局限，简化为"随机 + 模式"策略
- 方案 B：引入外部特征（销售额、奖池、季节）
- 方案 C：使用深度学习捕捉非线性模式（需升级 Python）

**优先级：** 🟢 低（根本性问题，难以解决）

---

### 5. 资金管理策略单一

**问题定位：**
- 当前策略：固定 2 注/期（¥4）
- 未根据中奖情况动态调整
- 连续未中奖时仍保持相同投入

**数据分析：**
- 最长连败记录：约 50 期未中奖
- 最大回撤：52.68%（发生在 2026 年初连续未中奖期间）

**优化方案：**
- 动态投注：连胜时增加，连败时减少
- 止损机制：亏损 70% 时暂停
- 止盈机制：盈利 50% 时取出利润

**优先级：** 🟡 中（不提升收益率，但降低风险）

---

### 6. 缺乏前向验证（Walk-Forward Validation）

**问题定位：**
- 当前回测：历史数据回测
- 未进行前向验证（训练集/测试集分离）
- 存在过拟合风险

**优化方案：**
```
训练集：2015-2022 年（1200 期）
测试集：2023-2026 年（508 期）
```
- 在训练集上优化参数
- 在测试集上验证效果
- 避免过拟合历史数据

**优先级：** 🟡 中（提高回测可信度）

---

### 7. 特征工程不足

**当前特征：**
- 冷热号频率（140+ 维）
- 遗漏值（47 维）
- 统计特征（和值、奇偶比等）

**缺失特征：**
- 期号特征（年初/年中/年末）
- 季节特征（节假日效应）
- 奖池特征（奖池高低影响）
- 销售额特征（热门程度）
- 历史同期特征（年同比）

**优化方案：**
- 增加时间特征（月份、季度、节假日）
- 增加市场特征（销售额、奖池）
- 增加模式特征（AC 值、区间比、尾数和）

**优先级：** 🟢 低（需外部数据支持）

---

### 8. 缺乏集成策略

**当前策略：**
- 单一 ML 模型预测
- 2 注/期固定输出

**优化方案：**
- 多模型集成：ML + 统计 + 随机
- 投票机制：3 模型投票选号
- 加权平均：不同策略权重分配

```
第 1 注：ML 模型（热号优先）
第 2 注：统计策略（遗漏 + 连号）
第 3 注（可选）：随机策略（博冷）
```

**优先级：** 🟡 中（预计可提升稳定性）

## ⚠️ 根本性限制

### 彩票的数学本质

**核心事实：**
- 彩票是独立随机事件
- 历史数据不影响未来结果
- 数学期望为负（返奖率 50-65%）

**推论：**
- 任何策略无法扭转负期望
- 最优策略是"控制成本，娱乐为主"
- 回测收益率 -70% 已接近理论极限

### 理性预期

**合理目标：**
- 收益率：-70% 至 -60%（已是最优）
- 中奖率：12-15%（符合概率）
- 成本控制：¥4/期（已优化）

**不合理目标：**
- 收益率 > 0%（不可能）
- 中奖率 > 30%（违背概率）
- 预测大奖（概率 1/1700 万）
