# ML 模型优化完成报告

## ✅ 已完成的 4 项优化

### 1️⃣ 增加随机扰动（ε-greedy 策略）

**问题：** 模型过度保守，只选热号

**解决：**
- 70% 概率按权重选号（利用模式）
- 30% 概率反向选冷号（探索模式）
- 避免过度拟合历史数据

**代码变化：**
```python
def predict(epsilon=0.3):
    if random.random() < epsilon:
        return explore_mode()  # 选冷号
    else:
        return exploit_mode()  # 选热号
```

---

### 2️⃣ 动态窗口（10 期 +30 期双窗口）

**问题：** 单一窗口无法同时捕捉短期趋势和长期规律

**解决：**
- **短期窗口（10 期）**：捕捉近期热号，快速响应
- **长期窗口（30 期）**：捕捉整体规律，抗噪声
- **对比特征**：识别升温/降温号码

**新增特征：**
- `short_front_freq_N`：短期频率
- `long_front_freq_N`：长期频率
- `freq_diff_N`：频率差异（升温/降温指标）
- `heating_count`：升温号码数量
- `cooling_count`：降温号码数量

---

### 3️⃣ 遗漏值加权（>15 期额外加分）

**问题：** 冷号遗漏很久但未得到额外关注

**解决：**
- 前区遗漏>15 期：权重 x2.0
- 前区遗漏>25 期：权重 x3.0
- 后区遗漏>8 期：权重 x2.0
- 后区遗漏>15 期：权重 x3.0

**策略逻辑：**
```
基础概率（冷号优先）
  ↓
遗漏检测
  ↓
>15 期 → x2.0
  ↓
>25 期 → 再 x1.5
```

---

### 4️⃣ 连号模式识别

**问题：** 模型生成的号码过于分散，不符合真实开奖

**解决：**
- 分析历史连号出现率（约 60%）
- 50% 概率生成 1 组连号
- 30% 概率生成 2 组连号
- 后区 35% 概率生成同尾号

**示例：**
- 连号：[12,13,25,28,31] ← 12-13 连号
- 同尾：[03,13] 或 [05,25] ← 尾数相同
