## 🔍 连号率问题分析

### 连号检测逻辑

```python
# 检测是否有连号
has_consecutive = any(pred_front[j+1] - pred_front[j] == 1 for j in range(len(pred_front)-1))
```

**问题：**
- 生成 1 组连号（如 12,13）+ 3 个随机号（如 5,20,30）
- 排序后：[5, 12, 13, 20, 30]
- 检测：12-13 是连号 → has_consecutive = True ✅

**但统计显示仅 49.7%：**
- 可能原因 1：连号生成逻辑仍有 bug
- 可能原因 2：ε=0.08 后，探索模式（无连号）占比增加
- 可能原因 3：连号生成失败时的回退逻辑
