# 彩票智能推荐系统 - 策略详解

> **版本：** V5.0  
> **最后更新：** 2026-03-12  
> **目标读者：** 用户、开发者

---

## 📋 概述

本系统使用 **7 种基础策略** + **3 种集成方法** + **自适应优化** 生成推荐号码。

**核心原则：**
- 不预测：明确告知 AI 不能预测中奖号码
- 有依据：所有推荐都有数据分析支撑
- 可解释：每种策略都有清晰的逻辑
- 透明：回测数据公开可查

---

## 🎯 7 种基础策略

### 策略 1：热号优先

**思路：** 选择历史上出现频率高的号码

**逻辑：**
```python
# 统计近 50 期各号码出现次数
# 选择出现次数最多的号码
# 前区选 5 个，后区选 2 个
```

**适用场景：** 常规投注，追求稳定

**风险：** 低

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 10.2% |
| 收益率 | -71.5% |
| 连号率 | 48.3% |

**推荐配置：**
- 注数：1-2 注
- 适合：新手、稳健型用户

**示例：**
```
前区热号：05 12 18 27 33（近 50 期出现 8-12 次）
后区热号：07 11（近 50 期出现 10-11 次）
推荐：05 12 18 27 33 + 07 11
```

---

### 策略 2：冷号反弹

**思路：** 选择很久没出现的号码（遗漏值高）

**逻辑：**
```python
# 计算每个号码的遗漏值（连续未出现期数）
# 选择遗漏值最高的号码
# 相信"该出了"的心理
```

**适用场景：** 博冷，追求高回报

**风险：** 高

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 8.4% |
| 收益率 | -78.2% |
| 连号率 | 35.6% |

**推荐配置：**
- 注数：1 注（博冷风险高）
- 适合：激进型用户

**示例：**
```
前区冷号：03 09 14 22 31（遗漏 25-40 期）
后区冷号：04 09（遗漏 30-35 期）
推荐：03 09 14 22 31 + 04 09
```

**⚠️ 风险提示：**
冷号可能继续冷，不要重仓博冷！

---

### 策略 3：连号组合

**思路：** 包含 1-2 组连号（如 12、13）

**逻辑：**
```python
# 统计历史连号出现频率
# 发现 52.4% 的开奖包含连号
# 生成包含连号的组合
```

**适用场景：** 提高连号率

**风险：** 中

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 9.8% |
| 收益率 | -72.3% |
| 连号率 | 85.2% |

**推荐配置：**
- 注数：1-2 注
- 适合：相信"连号运"的用户

**示例：**
```
连号组合 1：08 09 17 25 34（08-09 连号）
连号组合 2：12 13 19 27 33（12-13 连号）
推荐：08 09 17 25 34 + 06 12
```

**💡 数据支撑：**
历史数据显示，52.4% 的开奖包含连号，但此策略连号率 85.2%，过度拟合。

---

### 策略 4：奇偶均衡

**思路：** 奇偶比例保持 3:2 或 2:3

**逻辑：**
```python
# 统计历史奇偶比例
# 发现 3:2 和 2:3 占比最高（约 65%）
# 生成奇偶均衡的组合
```

**适用场景：** 均衡策略，分散风险

**风险：** 低

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 10.5% |
| 收益率 | -70.8% |
| 连号率 | 50.1% |

**推荐配置：**
- 注数：1-2 注
- 适合：稳健型用户

**示例：**
```
奇数：05 17 27 33（4 个）
偶数：12 18（2 个）
奇偶比：4:2（接近 3:2）
推荐：05 12 17 18 27 33 + 07 11
```

**💡 数据支撑：**
历史奇偶比分布：
- 3:2 → 32.5%
- 2:3 → 31.8%
- 4:1 → 18.2%
- 1:4 → 17.5%

---

### 策略 5：区间分布

**思路：** 将号码分成 5 个区间，每区间选 1 码

**逻辑：**
```python
# 将前区 35 个号码分成 5 区间
# 区间 1: 01-07
# 区间 2: 08-14
# 区间 3: 15-21
# 区间 4: 22-28
# 区间 5: 29-35
# 每区间选 1 个号码
```

**适用场景：** 分散风险，覆盖全面

**风险：** 低

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 11.2% |
| 收益率 | -68.5% |
| 连号率 | 42.3% |

**推荐配置：**
- 注数：1-2 注
- 适合：分散风险型用户

**示例：**
```
区间 1（01-07）：05
区间 2（08-14）：12
区间 3（15-21）：18
区间 4（22-28）：27
区间 5（29-35）：33
推荐：05 12 18 27 33 + 07 11
```

**💡 优势：**
回测收益率 -68.5%，是所有策略中最好的！

---

### 策略 6：遗漏加权

**思路：** 遗漏值高的号码权重高，但不只选冷号

**逻辑：**
```python
# 计算每个号码的遗漏值
# 遗漏值 > 20 → 权重 x1.3
# 遗漏值 > 30 → 权重 x1.5
# 遗漏值 < 5 → 权重 x0.8
# 按权重随机选择
```

**适用场景：** 捕捉冷号反弹

**风险：** 中

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 9.5% |
| 收益率 | -73.6% |
| 连号率 | 47.8% |

**推荐配置：**
- 注数：1 注
- 适合：想博冷但不想全冷的用户

**示例：**
```
高遗漏（权重 x1.5）：03 09
中遗漏（权重 x1.0）：14 22
低遗漏（权重 x0.8）：31
推荐：03 09 14 22 31 + 04 09
```

---

### 策略 7：自适应混合

**思路：** 动态调整上述 6 种策略的权重

**逻辑：**
```python
# 追踪近 10 期各策略表现
# 表现好的策略权重高
# 表现差的策略权重低
# 每期动态调整
```

**适用场景：** 智能投注，懒得选策略

**风险：** 中

**回测表现（1708 期）：**
| 指标 | 数值 |
|------|------|
| 中奖率 | 11.3% |
| 收益率 | -81.4% |
| 连号率 | 76.3% |

**推荐配置：**
- 注数：2-3 注
- 适合：相信 AI 动态调整的用户

**⚠️ 注意：**
虽然中奖率最高（11.3%），但收益率最差（-81.4%），因为过度拟合历史数据。

## 🔀 3 种集成方法

### 方法 1：Voting（投票）

**机制：**
1. 7 种策略各生成 10 注
2. 统计每个号码出现的频率
3. 选出频率最高的号码组合

**优势：**
- 综合所有策略
- 避免单一策略偏差
- 稳定性好

**劣势：**
- 计算量大
- 可能中庸

**适用：** 追求稳定性

**示例：**
```
策略 1 推荐：05 12 18 27 33
策略 2 推荐：03 09 14 22 31
策略 3 推荐：08 09 17 25 34
...
统计后：09 出现 5 次，12 出现 4 次，18 出现 4 次...
最终推荐：09 12 18 27 33 + 07 11
```

---

### 方法 2：Weighted（加权）

**机制：**
1. 按回测权重分配
2. 表现好的策略贡献更多号码
3. 表现差的策略贡献较少

**权重计算：**
```python
权重 = 50% × 收益率 + 30% × 中奖率 + 20% × 稳定性
```

**当前权重：**
| 策略 | 权重 |
|------|------|
| 区间分布 | 23% |
| 热号优先 | 20% |
| 奇偶均衡 | 17% |
| 连号组合 | 14% |
| 遗漏加权 | 12% |
| 冷号反弹 | 8% |
| 自适应混合 | 6% |

**优势：**
- 兼顾历史表现和多样性
- 向优秀策略倾斜

**劣势：**
- 依赖历史数据
- 可能错过新机会

**适用：** 相信历史数据

---

### 方法 3：Stacking（堆叠）

**机制：**
1. 分析当前数据特征
2. 选择最适合的策略组合
3. 动态调整

**特征判断：**
```python
if 近期连号多:
    选择 连号组合 + 热号优先
elif 近期冷号频繁:
    选择 冷号反弹 + 遗漏加权
else:
    选择 区间分布 + 奇偶均衡
```

**优势：**
- 动态适应数据特征
- 灵活性强

**劣势：**
- 复杂度高
- 可能过度拟合

**适用：** 特征明显时

## 💡 推荐配置

### 新手配置

**目标：** 稳定、风险低

```
策略：区间分布 + 热号优先
注数：2 注
预算：¥4 元/期
```

**理由：**
- 两个策略风险都低
- 收益率最好
- 简单易懂

---

### 进阶配置

**目标：** 平衡风险和收益

```
策略：区间分布 + 热号优先 + 奇偶均衡
注数：3 注
预算：¥6 元/期
```

**理由：**
- 三个策略互补
- 分散风险
- 中奖率更高

---

### 激进配置

**目标：** 追求高回报

```
策略：连号组合 + 遗漏加权 + 冷号反弹
注数：3 注
预算：¥6 元/期
```

**理由：**
- 博冷 + 博连号
- 风险高但可能中大奖
- ⚠️ 不建议长期使用

---

### 智能配置（推荐）

**目标：** AI 动态调整

```
策略：集成投票（Weighted）
注数：2-3 注（动态）
预算：¥4-6 元/期（动态）
```

**理由：**
- AI 根据历史表现调整
- 自动优化
- 省心

**⚠️ 注意：**
虽然智能，但回测表现不如固定策略（区间分布）。

## ⚠️ 风险提示

### 策略局限

1. **不能预测** - 所有策略都不能预测中奖号码
2. **历史不代表未来** - 回测结果仅供参考
3. **过度拟合风险** - 自适应策略可能过度拟合

### 投注建议

| 用户类型 | 推荐策略 | 注数 | 预算 |
|----------|----------|------|------|
| 新手 | 区间分布 | 1-2 注 | ¥2-4 元 |
| 稳健 | 区间分布 + 热号 | 2 注 | ¥4 元 |
| 进阶 | 集成投票 | 2-3 注 | ¥4-6 元 |
| 激进 | 连号 + 冷号 | 1-2 注 | ¥2-4 元 |

### 止损线

**建议：** 亏损 70% 时停止投注

```
初始资金：¥10,000
止损线：¥3,000（亏损 70%）
当前资金：< ¥3,000 → 停止投注
```
