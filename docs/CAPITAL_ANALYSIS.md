# 资金曲线分析系统

## 🎯 功能概述

自动分析彩票投注资金曲线，计算收益率、最大回撤，给出投注建议。

**核心指标：**
- 当前资金
- 累计收益率
- 最大回撤
- 风险级别
- 投注建议

---

## 📁 文件说明

| 文件 | 路径 | 说明 |
|------|------|------|
| 分析脚本 | `lottery/capital_analysis.py` | 生成资金分析报告 |
| 初始化脚本 | `lottery/init_capital_analysis.py` | 从历史数据回测生成初始资金记录 |
| 数据文件 | `lottery/stats/capital_stats.json` | 资金曲线历史数据 |
| 日志 | `lottery/logs/capital_analysis.log` | 运行日志 |

---

## 🚀 快速开始

### 首次使用（初始化数据）

```bash
cd /root/.openclaw/workspace/lottery
python3 init_capital_analysis.py
```

**说明：** 从历史开奖和推荐数据回测，生成初始资金曲线。

### 查看资金分析

```bash
python3 capital_analysis.py
```

## 📊 风险级别

| 级别 | 标识 | 条件 | 说明 |
|------|------|------|------|
| 🔴 危险 | 🟡 | 最大回撤 > 70% 或 ROI < -70% | 资金严重亏损 |
| 🟡 关注 | 🟡 | 最大回撤 > 50% 或 ROI < -50% | 需要警惕 |
| 🟢 安全 | 🟢 | 其他 | 资金健康 |

---

## 💡 投注建议

| 情况 | 建议 |
|------|------|
| 资金 < 初始 30% | ⚠️ 建议暂停投注，等待资金恢复 |
| 资金 < 初始 50% | 💡 建议降低投注，每期¥2-4 元 |
| 最大回撤 > 50% | 💡 建议保守投注，每期¥4-6 元 |
| 其他 | ✅ 可正常投注，每期¥6-10 元 |

## 🔄 数据积累流程

```
历史开奖数据 + 历史推荐数据
         ↓
  init_capital_analysis.py（回测）
         ↓
  capital_stats.json（初始数据）
         ↓
  capital_analysis.py（定期分析）
         ↓
  资金报告 + 投注建议
```

---

## 📝 注意事项

1. **首次使用必须先初始化**：运行 `init_capital_analysis.py`
2. **回测期数**：默认回测最近 30 期（受历史数据限制）
3. **简化奖级**：使用简化奖级表计算奖金（实际奖金可能浮动）
4. **自动更新**：定时任务每周日自动执行分析
