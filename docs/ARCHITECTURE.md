## 📋 概述

### 系统定位

**AI 驱动的彩票分析系统** - 支持大乐透、双色球、福彩 3D

- 数据采集 → 特征分析 → 策略生成 → 推荐推送 → 开奖核对
- 全自动定时任务，钉钉消息推送
- 透明、可解释、不忽悠

### 核心原则

1. **不预测** - 明确告知用户 AI 不能预测中奖号码
2. **有依据** - 所有推荐都有数据分析支撑
3. **会控制** - 预算提醒、止损线、强制停止
4. **全透明** - 代码开源、数据可查、逻辑可解释

## 🔄 数据流

### 推荐生成流程

```
1. 定时任务触发（每天 22:30）
        ↓
2. ml_predictor.py 启动
        ↓
3. lottery_storage.py 加载历史数据
        ↓
4. feature_analyzer.py 分析特征
   - 冷热号统计
   - 遗漏值计算
   - 连号检测
   - 奇偶比分析
        ↓
5. lottery_strategies.py 生成推荐
   - 7 种策略各生成 1-5 注
   - 每注标注策略类型
        ↓
6. adaptive_strategy.py 优化
   - 动态调整权重
   - 置信度评估
        ↓
7. push_best_strategy.py 推送
   - 格式化消息
   - 钉钉发送
        ↓
8. 保存到 reports/prediction.json
```

### 开奖核对流程

```
1. 定时任务触发（开奖后 15 分钟）
        ↓
2. auto_update_lottery.py / dlt_update_and_check.py
        ↓
3. 爬取官网开奖结果
        ↓
4. lottery_storage.py 保存新数据
        ↓
5. check_prize.py 核对
   - 读取最新推荐
   - 对比开奖号码
   - 计算中奖奖级
        ↓
6. 更新统计数据
   - 策略效果追踪
   - 中奖记录保存
        ↓
7. 钉钉推送核对结果
```

## 📦 模块说明

### 核心模块

#### ml_predictor.py

**职责：** ML 预测核心，生成推荐号码

**主要函数：**
- `load_model()` - 加载训练好的模型
- `analyze_features()` - 特征分析
- `generate_prediction()` - 生成预测
- `save_prediction()` - 保存结果

**依赖：**
- lottery_storage.py
- lottery_strategies.py
- feature_analyzer.py

**调用方式：**
```bash
python3 ml_predictor.py
```

---

#### lottery_storage.py

**职责：** 数据存储与读取

**主要功能：**
- 读取历史开奖数据（JSON）
- 保存新开奖结果
- 数据校验（期号连续性、号码范围）
- 增量更新

**数据结构：**
```json
{
  "期号": "2026028",
  "开奖日期": "2026-03-11",
  "前区": [5, 12, 18, 27, 33],
  "后区": [7, 11],
  "和值": 95,
  "跨度": 28
}
```

---

#### lottery_strategies.py

**职责：** 7 种基础策略实现

**策略列表：**
1. 热号优先 - 选择高频出现的号码
2. 冷号反弹 - 选择遗漏久的号码
3. 连号组合 - 包含 1-2 组连号
4. 奇偶均衡 - 奇偶比例 3:2
5. 区间分布 - 5 区间各选 1 码
6. 遗漏加权 - 遗漏值高的权重高
7. 自适应混合 - 动态调整上述策略

**接口：**
```python
def generate(strategy_name, num_sets=1):
    """生成推荐号码"""
    pass
```

---

#### backtest_v2.py

**职责：** 回测系统

**功能：**
- 历史区间回测（100-1708 期）
- 奖级计算（完整奖级表）
- 收益率计算（ROI、净收益）
- 中奖率统计
- 连号率统计
- 最大回撤计算
- 奖级分布统计

**输出：**
```json
{
  "version": "v3.1",
  "periods": 1708,
  "roi": -69.99,
  "win_rate": 10.60,
  "max_drawdown": 47.83,
  "total_investment": 341600,
  "total_prize": 102510
}
```

### 策略优化模块

#### adaptive_strategy.py

**职责：** 自适应策略 v4.0

**核心逻辑：**
- 动态 ε-greedy（0.02-0.15）
- 动态连号概率（60-90%）
- 动态投注注数（1-3 注）
- 状态追踪（近 20 期）
- 连胜/连败检测

**状态变量：**
```python
{
  "recent_results": [...],  # 近 20 期结果
  "win_streak": 0,          # 连胜期数
  "lose_streak": 0,         # 连败期数
  "peak_capital": 10000,    # 峰值资金
  "current_capital": 6500   # 当前资金
}
```

---

#### ensemble_strategy.py

**职责：** 集成策略

**三种集成方法：**

1. **Voting（投票）**
   - 7 种策略各生成 10 注
   - 统计号码出现频率
   - 选出高频号码组合

2. **Weighted（加权）**
   - 按回测权重分配
   - 表现好的策略贡献更多

3. **Stacking（堆叠）**
   - 根据特征分析选择最优组合
   - 动态适应数据特征

---

### 数据分析模块

#### feature_analyzer.py

**职责：** 特征工程

**分析特征：**
| 特征 | 说明 | 用途 |
|------|------|------|
| 冷热号 | 近 10/30/50 期频率 | 策略选择 |
| 遗漏值 | 号码连续未出现期数 | 冷号反弹 |
| 连号 | 相邻号码检测 | 连号组合 |
| 同尾号 | 同尾数号码（03/13/23） | 模式识别 |
| 奇偶比 | 奇数/偶数比例 | 均衡策略 |
| 和值 | 号码总和 | 趋势分析 |
| 跨度 | 最大号 - 最小号 | 分布分析 |
| 区间比 | 5 区间分布 | 分散风险 |

---

#### strategy_tracker.py

**职责：** 策略效果追踪

**功能：**
- 记录每期推荐
- 统计各策略胜率
- 生成效果报告
- 识别最优策略

**输出：**
```markdown
## 策略效果（近 30 期）

| 策略 | 中奖次数 | 中奖率 | 收益率 |
|------|----------|--------|--------|
| 热号优先 | 5 | 16.7% | -65% |
| 冷号反弹 | 2 | 6.7% | -82% |
| 连号组合 | 4 | 13.3% | -70% |
```

### 推送服务模块

#### push_best_strategy.py

**职责：** 钉钉推送

**流程：**
1. 读取最新推荐（reports/prediction.json）
2. 格式化消息（Markdown）
3. 调用钉钉机器人 API
4. 记录推送日志

**消息格式：**
```markdown
🎯 大乐透推荐（第 26029 期）

第 1 注：05 12 18 27 33 + 07 11 [热号优先]
第 2 注：03 09 14 22 31 + 04 09 [冷号反弹]
第 3 注：08 09 17 25 34 + 06 12 [连号组合]

💡 建议：每期投注不超过 2 注
⚠️ 提示：彩票有风险，理性投注
```

---

#### check_prize.py

**职责：** 中奖核对

**流程：**
1. 读取最新开奖结果
2. 读取上期推荐
3. 对比号码

5. 更新统计数据
6. 推送核对结果

        return "一等奖"
    elif red_matches == 5 and blue_matches == 1:
        return "二等奖"
