#### 2. 遗漏权重策略调整

**问题：**
- 当前遗漏>12 期 x1.3，>20 期 x1.2
- 回测显示遗漏与中奖呈负相关（越冷越难中）

**数据分析：**
```
遗漏 12-15 期号码：中奖概率约 11%
遗漏 16-20 期号码：中奖概率约 10%
遗漏 20+ 期号码：中奖概率约 8%
```

**优化方案：**
```python
# 方案 A：完全移除遗漏加权
if omission > 12:
    front_probs[i] *= 1.0  # 不加分

# 方案 B：改为"热号优先"
if frequency > 5:  # 热号
    front_probs[i] *= 1.5
elif omission > 15:  # 冷号
    front_probs[i] *= 0.8  # 降权

# 方案 C：遗漏反转（冷号继续冷）
if omission > 20:
    front_probs[i] *= 0.5  # 极冷号继续冷
```

**预期效果：** 收益率 +1-2%

**实施难度：** 低
**预计时间：** 20 分钟

# 3 模型投票
def ensemble_predict():
    pred1 = ml_frequency_model()  # 频率模型
    pred2 = ml_omission_model()   # 遗漏模型
    pred3 = ml_random_model()     # 随机模型

    votes = Counter()
    for pred in [pred1, pred2, pred3]:
        votes.update(pred['front'])
