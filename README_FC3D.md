# 快三彩票系统使用说明

## 📋 概述

彩票智能推荐系统已新增**快三**彩票类型支持，包含以下功能：

- ✅ 5 种策略生成推荐号码
- ✅ 自动中奖核对
- ✅ 策略效果追踪
- ✅ 双周汇总报告

---

## 🎲 快三游戏规则

### 基本玩法

快三游戏使用 3 个骰子，每个骰子 1-6 点，共 216 种组合。

### 投注类型与奖金

| 投注类型 | 中奖条件 | 奖金（元） |
|---------|---------|-----------|
| 三同号单选 | 指定豹子号完全匹配 | 240 |
| 三同号通选 | 开出任意豹子号 | 40 |
| 二同号单选 | 指定对子 + 单号完全匹配 | 80 |
| 二同号复选 | 指定对子出现 | 15 |
| 三不同号 | 三个号各不相同且完全匹配 | 40 |
| 二不同号 | 指定两个不同号出现 | 8 |
| 三连号通选 | 开出 123/234/345/456 | 10 |
| 和值 4/17 | 三数和为 4 或 17 | 80 |
| 和值 5/16 | 三数和为 5 或 16 | 40 |
| 和值 6/15 | 三数和为 6 或 15 | 25 |
| 和值 7/14 | 三数和为 7 或 14 | 16 |
| 和值 8/13 | 三数和为 8 或 13 | 12 |
| 和值 9/12 | 三数和为 9 或 12 | 10 |
| 和值 10/11 | 三数和为 10 或 11 | 9 |

## 🚀 使用方式

### 1. 生成推荐号码

## 📊 策略说明

### 5 种推荐策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 热号追踪 | 倾向于出现频率高的号码（1 和 6 权重高） | 追热趋势 |
| 冷号反弹 | 倾向于出现频率低的号码（3 和 4 权重高） | 赌概率回归 |
| 和值趋势 | 目标和值设定为中间区域（9-12） | 稳健型 |
| 形态追踪 | 随机选择形态（三同号/二同号/三连号/三不同号） | 形态投注 |
| 均衡策略 | 完全随机生成 | 随机选号 |

## 📝 数据格式

### 历史数据格式 (ks3_history.json)

```json
{
  "lottery_type": "ks3",
  "name": "快三",
  "records": [
    {
      "issue": "20260309001",
      "draw_date": "2026-03-09",
      "numbers": [1, 2, 3],
      "sum": 6
    }
  ]
}
```

### 推荐数据格式 (ks3_recommend.json)

```json
{
  "generated_at": "2026-03-09T19:50:00+08:00",
  "lottery_type": "ks3",
  "recommendations": [
    {
      "numbers": [1, 2, 3],
      "sum": 6,
      "pattern": "三连号",
      "strategy": "pattern_follow",
      "strategy_cn": "形态追踪"
    }
  ]
}
```

## ⚠️ 注意事项

1. **历史数据导入**
   - 当前快三历史数据为空
   - 需要导入实际开奖数据才能进行准确的热号/冷号分析
   - 可从官方渠道获取历史数据后导入

2. **奖金标准**
   - 使用标准奖金计算（非派奖期间）
   - 实际奖金以官方公布为准

3. **风险提示**
   - 彩票有风险，购买需谨慎
   - 策略分析仅供参考，不保证中奖
