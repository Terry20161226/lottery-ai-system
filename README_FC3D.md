# 福彩 3D 彩票系统使用说明

## 📋 概述

彩票智能推荐系统已新增**福彩 3D**彩票类型支持，包含以下功能：

- ✅ 5 种策略生成推荐号码
- ✅ 自动中奖核对
- ✅ 策略效果追踪
- ✅ 双周汇总报告

---

## 🎲 福彩 3D 游戏规则

### 基本玩法

福彩 3D 游戏使用 3 个号码，每个号码 0-9，共 1000 种组合。

### 投注类型与奖金

| 投注类型 | 中奖条件 | 奖金（元） |
|---------|---------|-----------|
| 直选 | 号码 + 顺序完全匹配 | 1040 |
| 组三 | 两个号码相同（顺序不限） | 346 |
| 组六 | 三个号码都不同（顺序不限） | 173 |

## 📊 策略说明

### 5 种推荐策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 热号追踪 | 倾向于出现频率高的号码 | 追热趋势 |
| 冷号反弹 | 倾向于出现频率低的号码 | 赌概率回归 |
| 和值趋势 | 目标和值设定为中间区域（9-14） | 稳健型 |
| 形态追踪 | 随机选择形态（豹子/组三/组六） | 形态投注 |
| 均衡策略 | 完全随机生成 | 随机选号 |

## 📝 数据格式

### 历史数据格式 (fc3d_history.json)

```json
{
  "lottery_type": "fc3d",
  "name": "福彩 3D",
  "records": [
    {
      "issue": "2026057",
      "draw_date": "2026-03-08",
      "numbers": [4, 7, 7],
      "sum": 18
    }
  ]
}
```

### 推荐数据格式 (fc3d_recommend.json)

```json
{
  "generated_at": "2026-03-09T20:45:00+08:00",
  "lottery_type": "fc3d",
  "recommendations": [
    {
      "numbers": [9, 5, 0],
      "sum": 14,
      "pattern": "组六",
      "strategy": "hot_number",
      "strategy_cn": "热号追踪"
    }
  ]
}
```

## ⚠️ 注意事项

1. **历史数据导入**
   - 当前快三历史数据为空
   - 需要导入实际开奖数据才能进行准确的热号/冷号分析
   - 可从官方渠道获取历史数据后导入

2. **奖金标准**
   - 使用标准奖金计算（非派奖期间）
   - 实际奖金以官方公布为准

3. **风险提示**
   - 彩票有风险，购买需谨慎
   - 策略分析仅供参考，不保证中奖
