## 📝 写作模板

### 开场（故事化）

```
【时间】+【场景】+【对话/心理活动】

例：
第 3 天晚上 21:30，我盯着手机屏幕。
开奖了。
"中了！"我喊了一声。
老婆跑过来："中多少？"
"5 块。"
她翻了个白眼："至于吗？"
至于。因为这是 AI 第一次帮我选号。
```

### 主体（数据 + 故事）

```
1. 发生了什么（故事）
2. 数据怎么说（分析）
3. 我怎么想（感受）
4. 给读者的建议（价值）
```

### 结尾（互动 + 预告）

```
【互动问题】+【下期预告】

例：
你在第几天开始相信 AI 的？
评论区聊聊。

下期：第 4 天，AI 说今天要"保守投注"。
```
