## 🎯 为什么要做资金分析？

### 之前的问题

之前我们只看"收益率"，比如：

- 策略 A：-70%
- 策略 B：-80%
- 策略 C：-69.99%

**但这些数字太抽象了。**

读者问："那我投 1000 块，实际亏多少？"

我答不上来。

### 现在的答案

有了资金曲线分析，可以回答：

- **当前资金：** ¥9,982.5
- **累计收益：** -¥17.5
- **最大回撤：** 0.2%
- **风险级别：** 🟢 安全

**翻译成人话：**

投 1 万块，目前亏 17 块 5。

---

## 📊 资金曲线是怎么来的？

### 数据来源

AI 从历史数据中回测：

1. **历史开奖数据** - 大乐透 24 期 + 双色球 50 期
2. **历史推荐数据** - 每期的推荐号码
3. **模拟投注** - 每期¥10 元（5 注）

### 计算过程

```
第 1 期：
  投入：¥10
  中奖：¥0
  资金：¥9,990

第 2 期：
  投入：¥10
  中奖：¥5
  资金：¥9,985

第 3 期：
  投入：¥10
  中奖：¥0
  资金：¥9,975

...

当前：
  资金：¥9,982.5
```

### 关键指标

| 指标 | 数值 | 说明 |
|------|------|------|
| **初始资金** | ¥10,000 | 模拟起始资金 |
| **当前资金** | ¥9,982.5 | 最新资金 |
| **累计收益** | -¥17.5 | 总盈亏 |
| **收益率** | -0.2% | 收益/本金 |
| **最大回撤** | 0.2% | 最大亏损幅度 |

---

## 🎛️ 风险级别

系统根据 2 个指标判断风险：

### 1. 最大回撤

**定义：** 从最高点跌到最低点的幅度

**当前：** 0.2%

**风险等级：**

| 回撤 | 级别 | 标识 |
|------|------|------|
| >70% | 危险 | 🔴 |
| >50% | 关注 | 🟡 |
| <50% | 安全 | 🟢 |

### 2. 累计收益率

**定义：** 总收益占本金的比例

**当前：** -0.2%

**风险等级：**

| 收益率 | 级别 | 标识 |
|--------|------|------|
| <-70% | 危险 | 🔴 |
| <-50% | 关注 | 🟡 |
| >-50% | 安全 | 🟢 |

### 综合判断

**当前：** 🟢 安全

**解读：** 资金波动很小，风险可控。

## ⚠️ 风险提示

### 1. 数据有限

**当前回测期数：** 仅 2 期

**建议：** 至少积累 30 期数据才有参考价值。

### 2. 简化奖级

系统使用简化奖级表计算奖金，实际奖金可能浮动。

**例如：**

- 一等奖：500 万（实际可能 300-1000 万）
- 九等奖：5 元（固定）

### 3. 历史不代表未来

回测数据是历史表现，不代表未来收益。

**彩票的本质：** 负期望游戏

**长期趋势：** 越买越亏

### 4. 理性投注

**建议：**

- 每期投注：¥6-10 元
- 每月建议上限：¥500 元
- 亏损 70% 时：建议停止

## 📝 常见问题

### Q1：为什么收益率是负的？

**因为彩票是负期望游戏。**

从数学上讲，彩票的返奖率约 50-65%，意味着：

- 投入 100 块
- 平均返还 50-65 块
- 长期必亏

**这是彩票的设计机制，不是 AI 的问题。**

### Q2：资金分析有什么用？

**帮你了解真实盈亏。**

很多人买彩票没有记账，不知道实际亏了多少。

资金分析可以：

- 记录每期投入和收益
- 计算累计盈亏
- 判断风险级别
- 给出投注建议

### Q3：多久更新一次？

**每周日 21:30 自动更新。**

你也可以手动运行：

```bash
python3 capital_analysis.py
```

### Q4：数据从哪来？

**从历史开奖和推荐数据回测。**

系统会：

1. 加载历史开奖数据
2. 加载历史推荐数据
3. 模拟每期投注
4. 计算中奖金额
5. 生成资金曲线
