**原因：**
- 直选概率：1/1000 = 0.1%
- 组选概率：约 9%（组三 + 组六）
- 当前策略未充分利用组选优势

### 2. 冷号反弹策略失效（ROI -100%）
**问题：** 冷号反弹策略 100 期未中奖
**原因：**
- 冷号定义过于简单（仅基于出现次数）
- 未考虑遗漏值动态变化
- 冷号可能继续冷

### 3. 投注策略过于分散
**问题：** 每期 6 注，成本¥12，但中奖覆盖不足
**原因：**
- 热号、冷号、均衡各 2 注，缺乏重点
- 未根据形态概率分配投注比例

### 4. 形态分析缺失
**问题：** 未利用组三/组六概率差异
**原因：**
- 组三概率 27%，组六概率 72%

# 原策略：直选为主
bets = generate_direct_bets()

# 优化后：组选为主
def generate_group_bets(hot_nums, cold_nums):
    """生成组选投注"""
    bets = []
    # 组六投注（概率 72%）
    for combo in combinations(hot_nums, 3):
        bets.append({'type': '组六', 'numbers': list(combo)})
    # 组三投注（概率 27%）

    return bets
