# 彩票智能推荐系统 v4.0

基于机器学习的彩票预测系统，支持大乐透和双色球。

**⚠️ 免责声明：** 本项目仅供学习和娱乐，不保证盈利。彩票是负期望游戏，请理性投注。

---

## 🎯 功能特性

- 📊 **历史数据采集** - 1708 期大乐透 +1449 期双色球
- 🤖 **ML 预测模型** - 统计频率 + 特征工程 + 自适应策略
- 🎲 **7+ 种策略** - 热号优先、冷号反弹、连号组合等
- 📈 **完整回测** - 支持 100-1708 期回测验证
- 🔄 **自适应策略 v4.0** - 动态 ε/连号/投注调整

## 🚀 快速开始

### 安装依赖

```bash
pip3 install requests numpy pandas
```

### 生成推荐

```bash
cd lottery
python3 ml_predictor.py
```

### 运行回测

```bash
cd lottery
python3 backtest_v2.py
```
