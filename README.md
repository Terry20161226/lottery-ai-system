## 🎯 核心功能

### 1️⃣ 数据采集与更新

| 彩种 | 数据量 | 更新频率 | 来源 |
|------|--------|----------|------|
| 大乐透 | 1708 期 | 每周一/三/六 21:30 | 体彩官网 |
| 双色球 | 1449 期 | 每周二/四/日 21:30 | 福彩官网 |
| 福彩 3D | 实时 | 每天 21:00 | 福彩官网 |

### 2️⃣ 智能预测

- 🤖 **ML 预测模型** - 基于历史数据的统计分析
- 🎲 **7 种策略** - 热号、冷号、连号、奇偶等
- 🔄 **自适应调整** - 根据近期表现动态优化
- 📊 **置信度评估** - 高/中/低置信度标注

### 3️⃣ 回测验证

- 📈 **历史回测** - 支持 100-1708 期回测
- 💰 **收益分析** - ROI、中奖率、最大回撤
- 📋 **奖级统计** - 各奖级中奖次数
- 📉 **资金曲线** - 可视化资金变化

### 4️⃣ 自动核对

- ✅ **开奖后自动核对** - 中奖情况自动统计
- 📊 **策略效果追踪** - 各策略胜率分析
- 📧 **钉钉推送** - 推荐和开奖结果自动推送
- 📅 **定时任务** - 全自动运行

## 📈 策略说明

### 7 种基础策略

| 策略 | 思路 | 适用场景 | 风险 |
|------|------|----------|------|
| 热号优先 | 选高频出现的号码 | 常规投注 | 低 |
| 冷号反弹 | 选很久没出的号码 | 博冷 | 高 |
| 连号组合 | 包含 1-2 组连号 | 提高连号率 | 中 |
| 奇偶均衡 | 奇偶比例 3:2 | 均衡策略 | 低 |
| 区间分布 | 5 区间各选 1 码 | 分散风险 | 低 |
| 遗漏加权 | 遗漏值高的权重高 | 捕捉反弹 | 中 |
| 自适应混合 | 动态调整策略 | 智能投注 | 中 |

### 推荐配置

**新手：** 热号优先 + 奇偶均衡（2 注）
**进阶：** 自适应混合（2-3 注）
**激进：** 冷号反弹 + 连号组合（1 注）
