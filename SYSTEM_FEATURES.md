## 📈 策略库

### 已实现策略（7+ 种）

| 策略名 | 说明 | 适用场景 |
|--------|------|----------|
| **热号优先** | 选择高频出现的号码 | 常规投注 |
| **冷号反弹** | 选择遗漏较久的号码 | 博冷 |
| **连号组合** | 包含 1-2 组连号 | 提高连号率 |
| **奇偶均衡** | 奇偶比 3:2 或 2:3 | 均衡策略 |
| **区间分布** | 5 区间各选 1 码 | 分散风险 |
| **遗漏加权** | 遗漏值加权选择 | 捕捉反弹 |
| **自适应混合** | 动态调整上述策略 | 智能投注 |
