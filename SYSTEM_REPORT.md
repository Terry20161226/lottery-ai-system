### 2. 定时任务优化（中优先级）

| 当前 | 建议 | 理由 |
|------|------|------|
| 每 10 分钟同步 | 每 30 分钟同步 | 减少 API 调用，降低 GitHub 限流风险 |
| 无开奖更新任务 | 添加开奖后自动更新 | 双色球 (二/四/日 21:30)、大乐透 (一/三/六 21:30) |
| 无心跳检测 | 添加系统健康检查 | 监控磁盘、进程、数据更新状态 |

**建议 crontab：**
```bash
# Git 同步（30 分钟）
*/30 * * * * cd /root/.openclaw/workspace && ./auto_sync.sh >/dev/null 2>&1

# 福彩 3D 推荐（每日 21:00）
0 21 * * * cd /root/.openclaw/workspace/lottery && ./fc3d_cron.sh

# 双色球开奖更新（二/四/日 21:45）
45 21 * * 2,4,0 cd /root/.openclaw/workspace/lottery && python3 auto_update_lottery.py

# 大乐透开奖更新（一/三/六 21:45）
45 21 * * 1,3,6 cd /root/.openclaw/workspace/lottery && python3 auto_update_lottery.py

# 每日推荐生成（22:30）
30 22 * * * cd /root/.openclaw/workspace/lottery && python3 ml_predictor.py

# 中奖核对（每日 23:00）
0 23 * * * cd /root/.openclaw/workspace/lottery && python3 check_prize.py
```
